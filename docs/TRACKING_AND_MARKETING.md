# Tracking and Marketing Setup

## Theme placeholders

Tracking placeholders are exposed in Shopify theme settings:

- Google Analytics Measurement ID
- Google Tag Manager Container ID
- Meta Pixel ID
- Klaviyo Company ID
- Google Ads Conversion ID

The snippets render only when an ID is present.

## Recommended launch sequence

1. Create Google Analytics 4 property.
2. Create Google Tag Manager container.
3. Create Meta Pixel and connect it to the Shopify sales channel or theme setting.
4. Create Klaviyo account, connect Shopify and add the company ID.
5. Configure Google Search Console and submit sitemap.
6. Configure Google Merchant Center after products, shipping and policies are ready.

## Conversion events

Track quote form submissions, contact submissions, artwork intake, add to cart, checkout started and purchases. Use GTM for event routing when possible so ad platforms can be adjusted without theme edits.
