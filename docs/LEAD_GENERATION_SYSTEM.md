# Lead Generation System

L F Graphco's Shopify theme includes a conversion system built around quote requests first and phone calls second.

## Primary Conversion Paths

- Homepage lead capture form for name, company, email, phone, project type, quantity and due date.
- Sticky mobile CTA with `Call 972-840-8337` and `Request Quote`.
- Exit-intent quote popup with a short Shopify contact form.
- Free mockup offer sections that encourage artwork upload and quote requests.
- Reusable trust sections for screen printing, embroidery, DTF printing and promotional products.

## Landing Pages

The theme includes focused page templates for:

- Team Stores
- School Spirit Wear
- Company Uniforms
- Church Apparel
- Fundraisers

Each landing page uses conversion-focused copy, trust messaging, a mockup offer, quote CTAs and a lead capture form.

## Shopify Forms

Forms use Shopify contact form fields so leads can flow through the store notification system without a custom backend. Lead fields include:

- `contact[name]`
- `contact[organization]`
- `contact[email]`
- `contact[phone]`
- `contact[quote_type]`
- `contact[quantity]`
- `contact[deadline]`

## Launch Notes

Create or assign Shopify pages using the matching templates, then add navigation links in Shopify Admin if handles differ from the default theme links.
